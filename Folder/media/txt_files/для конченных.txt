Лабораторная работа №1
Основы программирования на языке Ruby
Задания для самостоятельного выполнения (вариант брать по номеру в группе, включая остаток от деления на 10 плюс 1, если номер в группе больше 10):
Вариант 1:
Разработать программу, которая запрашивает у пользователя два целых числа, вычисляет их сумму и выводит результат в виде поясняющей строки.
Реализовать программу, определяющую знак введённого числа (отрицательное, положительное или ноль).
Составить программу, которая выводит все целые числа от 1 до 10 включительно с использованием цикла while.
Сформировать хеш, содержащий сведения о студенте (ФИО, возраст, номер группы). Вывести данные в виде форматированной строки.
Вариант 2:
Составить программу, вычисляющую разность двух введённых чисел и выводящую результат в виде поясняющей строки.
Реализовать программу, определяющую чётность введённого числа.
Составить программу, выводящую все чётные числа из диапазона 1–20 включительно с использованием цикла until.
Сформировать хеш, содержащий перечень товаров и их цены. Реализовать вывод стоимости заданного товара по ключу.
Вариант 3:
Составить программу, вычисляющую произведение двух введённых чисел. Результат отобразить в поясняющей строке.
Реализовать программу, проверяющую делимость числа на 3 без остатка.
Составить программу, выводящую все нечётные числа из диапазона 1–20.
Сформировать хеш, содержащий сведения о городе (название, страна, численность населения). Добавить новый ключ :year_founded (год основания) и вывести обновлённый хеш.
Вариант 4:
Составить программу, выполняющую деление одного введённого числа на другое и выводящую результат в строке «Результат деления: …». Обеспечить обработку случая деления на ноль.
Реализовать программу, определяющую делимость числа на 5 без остатка.
Составить программу, вычисляющую сумму всех чисел от 1 до N (N задаётся пользователем).
Сформировать хеш, содержащий сведения о книге (название, автор, год издания). Удалить один из ключей и вывести обновлённый хеш.
Вариант 5:
Составить программу, вычисляющую остаток от деления одного числа на другое.
Разработать программу, определяющую четверть координатной плоскости по введённым значениям x и y.
Составить программу, вычисляющую произведение всех чисел от 1 до N.
Сформировать хеш, содержащий сведения о стране (название, столица, население). Реализовать вывод всех ключей хеша.
Вариант 6:
Разработать программу, возводящую введённое число в квадрат и выводящую результат в поясняющей строке.
Составить программу, определяющую, является ли введённый символ гласной буквой русского алфавита.
Сформировать массив случайных чисел и определить максимальное значение среди его элементов.
Составить программу, выводящую все значения хеша в виде массива.
Вариант 7:
Разработать программу, возводящую число в степень, введённую пользователем.
Реализовать программу, определяющую возрастную категорию (ребёнок, подросток, взрослый) в зависимости от введённого возраста.
Сформировать массив случайных чисел и определить минимальное значение среди его элементов.
Написать программу, преобразующую строку "123" в число, прибавляющую к нему 10 и выводящую результат.
Вариант 8:
Составить программу, вычисляющую длину введённой строки.
Реализовать программу, определяющую название дня недели по его номеру (1–7).
Составить программу, определяющую количество элементов в массиве, введённом пользователем.
Составить программу, преобразующую число 456 в строку и объединяющую его с другой строкой.
Вариант 9:
Составить программу, преобразующую введённую строку в верхний регистр и выводящую результат.
Разработать программу, проверяющую корректность введённого пароля. При совпадении с "ruby123" вывести сообщение «Доступ разрешён», иначе — «Неверный пароль».
Составить программу, выводящую элементы массива в обратном порядке.
Разработать программу, преобразующую массив [1, 2, 3] в строку "1,2,3" и выводящую её.
Вариант 10:
Составить программу, разворачивающую строку и выводящую результат в поясняющей форме: «Обратная строка: …».
Реализовать программу, определяющую время суток (утро, день, вечер, ночь) по введённому часу (0–23).
Составить программу, умножающую каждый элемент массива на 2 и выводящую полученный массив.
Разработать программу, разбивающую строку "apple,banana,orange" в массив и выводящую количество элементов массива.
